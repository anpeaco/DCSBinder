# Icons

[Lucide](https://lucide.dev) SVG icons, [ISC license](https://github.com/lucide-icons/lucide/blob/main/LICENSE).
24×24 viewBox, embedded with `stroke="#fff"` so Slint's `colorize` property
can tint them at the call site.

| File | Lucide name | Used for |
|---|---|---|
| `refresh-cw.svg` | `refresh-cw` | Rescan button |
| `undo-2.svg` | `undo-2` | Undo-last button |
| `check.svg` | `check` | Confirm/apply |
| `x.svg` | `x` | Cancel / close |
| `align-left.svg` | `align-left` | Inline diff mode |
| `columns-2.svg` | `columns-2` | Side-by-side diff mode |
| `list-filter.svg` | `list-filter` | Changed-only toggle |
| `search.svg` | `search` | Filter box prefix |
| `circle-alert.svg` | `circle-alert` | DCS-running warning |
| `gamepad-2.svg` | `gamepad-2` | Live devices summary |
| `plane.svg` | `plane` | Aircraft summary |
| `layers.svg` | `layers` | Apply-scope = all aircraft |
| `target.svg` | `target` | Apply-scope = this aircraft only |
| `trash-2.svg` | `trash-2` | Discard stale file (moves to backup) |
